# Обновление товаров Casio на Ozon

Этот скрипт помогает управлять товарами Casio на площадке Ozon.

## Что делает скрипт

1. Скачивает актуальные остатки с сайта Casio.

2. Обновляет количество товаров в вашем магазине на Ozon в соответствии с реальными остатками.

3. Обновляет цены товаров, приводя их к корректному формату для Ozon.

4. Логирует ошибки, если что-то пошло не так, например проблемы с интернет-соединением.

## Для кого полезно

- Продавцы Casio на Ozon, которым нужно поддерживать актуальные цены и наличие товаров.

- Сотрудники, отвечающие за ассортимент, чтобы не тратить время на ручное обновление.

## Примечания

Скрипт работает автоматически с данными поставщика, поэтому цены и остатки всегда соответствуют реальному состоянию.

Он не требует вмешательства в код, просто загружает актуальные данные и синхронизирует магазин.
